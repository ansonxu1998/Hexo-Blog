STM32 定时器产生 PWM 波

对于 PWM 波来说最重要的两个参数就是频率和占空比。频率是周期的倒数，占空比是指脉宽时间占周期的比例。微控制器产生 PWM 波的原理是，定时器重复的从 0 计数到 ARR，设定比较值 CCRx。当定时器的计数值向上计数到与 CCRx 的值相等时，对应波形输出管脚电平翻转，当定时器计数值到 ARR 时，输出管脚的电平再次发生翻转。定时器的一个计时周期对应一个 PWM 周期，脉宽（高电平的宽度）需要根据输出管脚的初始电平不同，或者为 0~CCRx 这一段，或者为 CCRx~ARR 这一段。改变 CCRx 的值即可实现脉宽的控制。PWM 频率和占空比的影响因素有：

  • ARR：决定 PWM 周期（在系统时钟频率固定的情况下）
  • CCRx：决定 PWM 占空比（高低电平所占整个周期比例）

STM32 的 TIM1 ~ TIM8 定时器，除了 TIM6 和 TIM7 外的其他定时器都可以用来产生 PWM 输出。高级定时器 TIM1 和 TIM8 可以同时产生 7 路的 PWM 输出。而通用定时器可以同时产生 4 路 PWM。

以通用定时器为例，这些定时器有 4 个输出通道，每个通道都有一个捕获/比较寄存器，将寄存器，将寄存器值和计数器值比较，通过比较结果输出高低电平，实现 PWM 信号输出。

每个定时器只有一个计数器，但每个通道都有自己的捕获/比较寄存器，因此对于一个定时器来说，4 路输出的 PWM 频率（周期）是相同的，而不同通道的占空比可以是不同的。

PWM 配置：

  • Prescaler 预分频系数：等待多少个时钟周期增加一次计数
  • Counter Model 计数模式：向上、向下或者中间对齐等
  • Counter Period 计数周期：重新开始计数前所要计数的最大数量
  • Internal Clock Division 内部时钟分频：另一种时钟分频方法
  • auto-reload-preload 自动重装载：当计数达到最大时，重新启动计数器

Calculate the PWM frequency

  • System Clock is 72 MHz: We can "count" 72-million times each second
  • 8-bit resolution(0-255): Each cycle is 256 "counts"

\[ \frac{72,000,000\ \text{counts}}{1\ \text{second}}\cdot\frac{1\ \text{cycle}}{256\ \text{counts}}=\frac{218250\ \text{cycles}}{1\ \text{second}} \]

PWM frequency = 218250 Hz

  • Mode 模式：PWM1 、PWM2

    模式 计数器 CNT 计数方式 说明
    PWM1 递增 CNT<CCR，通道 CH 为有效，否则为无效
    递减 CNT>CCR，通道 CH 为无效，否则为有效
    PWM2 递增 CNT<CCR，通道 CH 为无效，否则为有效
    递减 CNT>CCR，通道 CH 为有效，否则为无效

  • Pulse 占空比：Counter Periode 之间的一个数
